# Dev Playbook

Universal software development framework for LLM-assisted projects.

**Working Language:** The official written working language for this project is **British English**.

---

## Overview
This repository acts as the **Single Source of Truth** for the team's development methodology, conventions, and tooling. It is designed to be managed and evolved by both humans and autonomous agents.

## Getting Started
1. Read `PROJECT_STATUS.md` for the current context.
2. Review `software-development-playbook.md` for the core methodology.
3. Consult `AI-AGENTS.md` and agent-specific files (`CLAUDE.md`, `GEMINI.md`) for operational rules.
